## 使用教學
1. 先下載這坨檔案 `Code > Download ZIP`
2. 先去裝 [`NodeJS`](https://nodejs.org/zh-tw/)
3. 然後開個終端機切到這個目錄下面打 `npm i`
4. 然後打 `node server.js`

## 網頁路徑
- `/`: 輸出頁
- `/video.html`: 影片頁面
- `/admin.html`: 管理頁面
